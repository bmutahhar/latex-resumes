As a Technical Contributor
- Take ownership of significant technical initiatives or product areas.
- Help refine frontend patterns across multiple frameworks (React and SvelteKit).
- Improve and maintain shared UI components and styling conventions using Tailwind.
- Optimize performance across build time, runtime, and edge delivery.
- Contribute to CI/CD workflows using GitHub and modern deployment pipelines.
- Collaborate with backend engineers to shape efficient APIs and edge-compatible architectures.
- Improve developer experience by enhancing local tooling, Vite configs, linting, testing setup, and documentation.

Requirements:
You are a strong hands-on engineer who thrives in modern frontend ecosystems and cares deeply about performance, developer experience, and clean architecture.
- Proven experience building production-grade applications with React and/or SvelteKit.
- Strong proficiency in modern JavaScript and TypeScript.
- Experience using Vite or similar modern build tools.
- Solid experience with Tailwind CSS or utility-first CSS approaches.
- An understanding of client-side rendering, SSR, and edge rendering patterns.
- Experience deploying or integrating applications with edge platforms (e.g., Cloudflare Workers or similar).
- Experience with Git-based workflows (GitHub), pull requests, and CI/CD pipelines.
- Good knowledge of web performance optimization (Core Web Vitals, bundle size, caching strategies).
- Experience writing automated tests (unit, integration, or e2e).
- Strong communication skills in English and experience working in cross-functional teams.
